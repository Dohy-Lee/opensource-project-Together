# 🙋🏻‍♂Together

- 특가상품으로 나오는 물품은 수량이 너무 많아 혼자 구매하기 어려운 경우가 많다. 특히나 대학가 주변 혼자 자취하는 대학생들에게는 더욱 부담일 수 있다. 우리는 이 "Together" 어플을 통해 내 주위 동네 사람들과 특가 상품을 공동구매하여 자신에게 필요한 물품을 필요한 만큼 보다 저렴한 가격으로 살 수 있다.
